The iPhone alarm interface requires the user to go through multiple pages of tasks, a scroll selector (type input box for the new update), and other annoyances just to create a basic alarm clock. My solution is to create something that you can make on one page and have everything selected with simple button presses. It is not perfect, but it is a working prototype that I think shows how much better it could be.
